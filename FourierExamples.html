Generating a Sample Signal with Varying Frequency

Example 1: Manual Spans, Manual Frequencies

clear;
Frecuencies = [20,80,120]; %Choose your own Frequencies
Spans = [0.3,0.5,0.3]; %Choose the Span of the Freqs in Secs
fs = max(Frecuencies)*2.5; %Select a Frequency Sample of 2.5 times the max frequency (Above Nyquist Theorem)
F =[]; % Generate a List for the Signal
iter = 1; %Iterate Through Spans
for freq = Frecuencies
F = [F,repelem(freq,Spans(iter)*fs)]; %Generate an array of frequencies and repeat N times depending on Span.
iter = iter + 1;
end
dt = 1/fs; %Generate Delta for signal sampling
StopTime = length(F)/fs; %Length of Signal (Sum of Spans)
t = (0:dt:StopTime-dt); %generate Array of times
amp = 1; %Amplitude of the Signal
 
 
for i= 1:length(t)
signal(i) = amp*sin(2*pi*F(i)*t(i)); %Generate Sinusoid
end;
 
figure() %generate a figure Avoid figures in cache
plot(t,signal)

Example 2: Same Span, Manual Freqs

clear;
Frecuencies = [1,5,10,15,20,30,50,100];
fs = max(Frecuencies)*2.5;
F = [];
for freq = Frecuencies
F = [F,repelem(freq,fs)]; %Frequency Array
end
 
dt = 1/fs; %Generate Delta for signal sampling
StopTime = length(F)/fs; %Length of Signal (Sum of Spans)
t = (0:dt:StopTime-dt); %generate Array of times
amp = 1; %Amplitude of the Signal
 
 
for i= 1:length(t)
signal(i) = amp*sin(2*pi*F(i)*t(i)); %Generate Sinusoid
end;
 
figure() %generate a figure Avoid figures in cache
plot(t/length(Frecuencies),signal)

Fourier Transform (FFT) and Short Time Fourier Transform

The Fourier Transform works in this way:

The Folding

will happen when the Sampling Frecuency is not at least 2 times larger than the maximum frecuency (Or the desired Frequency within the signal)
In this case for example at a sampling rate of 1 KHz if the maximum Frequency within signal is 600Hz we will obtain a Fourier Frequency component at 400 Hz because the FFT will work up to 500Hz Components.

clear all;
Fs = 1000; % Sampling frequency
T = 1/Fs; % Sampling period
L = 1000; % Length of signal
t = (0:L-1)*T; % Time vector
x1 = cos(2*pi*50*t); % First row wave
x2 = cos(2*pi*150*t); % Second row wave
x3 = cos(2*pi*300*t); % Third row wave
 
X = [x1; x2; x3];
for i = 1:3
subplot(3,1,i)
plot(t(1:100),X(i,1:100))
title(['Row ',num2str(i),' in the Time Domain'])
end
 
n = 2^nextpow2(L);
dim = 2;
Y = fft(X,n,dim);
P2 = abs(Y/L);
P1 = P2(:,1:n/2+1);
P1(:,2:end-1) = 2*P1(:,2:end-1);
for i=1:3
subplot(3,1,i)
plot(0:(Fs/n):(Fs/2-Fs/n),P1(i,1:n/2))
title(['Row ',num2str(i),' in the Frequency Domain'])
end
